Toxic Comment Classifier
  • Toxic comments Classifier is a web app based on NLP and Machine Learning in which you can enter your comment and check whether your comment is toxic or not.
  • It is a Multi Label Classifier, you can classify your comment in 6 different classes which are Toxic, Severe Toxic, Obscene, Threat, Insult, Identity Hate
  • Used NLP Preprocessing Techniques Like Contraction, Stopwords and Lemmatization, Tfidf Vectorization.
    Used 7 different Machine Learning Algorithms and 4 different Deep Learning Algorithms to train the model.
  • Build this web app using Flask, HTML, CSS, Bootstrap.
